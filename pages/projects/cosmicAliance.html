Cosmic aliance


What is this project about?
Cosmic Aliance is a 2D bullet hell game. The player has 3 ships that he uses at the same time, being able to switch between ships when ever he wants. Each ship has it's own health, ammo, skill and special ability. The player has to destroy all the enemies to win the game.
