Roc Gaem


What is this project about?
Roc gaem is a game based in the original arcade game Asteroids where the player controls a spaceship and tries to destroy the asteroids.
The game has a infinite amount of levels and the player can choose between various weapon types unlocked each level the player completes. There are 3 types of asteroids, each with a different behavior. Lastly the player counts with 2 different skills, one to teleport and one to slowdown the time speed.


What did I learn?
As this was my first Unity project and I had no previous experience with Unity, I learned how to code and use Unity. I also learned how to use the Post Processing system and how to use the Audio system.
